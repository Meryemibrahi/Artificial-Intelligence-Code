# Working with AI 

In recent years, AI and Machine Learning have been used to create amazing projects and improve many aspects of our lives. It has become essential to learn how these models work and to explore what they are capable of.  In this repository, you can find some of the AI projects that I have worked on. Each project explores a new aspect of Machine Learning.

# Projects in this Repository
  1. Limits of ANNs: 
     This project trains 5 separate neural networks (CNNs) on binary 25×25 matrix inputs to solve increasingly complex spatial reasoning tasks. Each problem tests a   different limit of what ANNs can learn.


  2. Reward Model Training: 
     A reward model that predicts human-assigned quality scores for Turkish question-answer pairs using text embeddings and machine learning. Ran 60 experiments across 4 transformer embedding models, 5 text configurations, and 3 regression algorithms using Python, Hugging Face, and scikit-learn on a GPU. Analyzed results to identify which combination of inputs and models best captures answer quality.
